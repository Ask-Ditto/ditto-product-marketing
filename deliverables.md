# PMM Deliverables from Ditto Studies

What Claude Code generates from each study type. A single 10-persona,
7-question study produces raw material for multiple deliverables.

---

## The One-Study-to-Many-Deliverables Map

```
One Ditto Study (10 personas, 7 questions, ~20 min)
    |
    |-> Positioning scorecard ................ 5 min
    |-> Competitive battlecard ............... 5 min
    |-> Messaging hierarchy .................. 5 min
    |-> Objection handling guide ............. 3 min
    |-> Customer quote bank .................. 3 min
    |-> One-pager / sales leave-behind ....... 5 min
    |-> Pitch narrative ...................... 10 min
    |-> Blog article draft ................... 10 min
    |-> Social media thread .................. 3 min
    |-> ROI framework ........................ 5 min
    |-> Demo script .......................... 5 min

Total: ~60 min from zero to complete PMM kit
Traditional: 3-6 weeks, $15-50K in agency fees
```

## Deliverable Details by Study Type

### From a Positioning Validation Study

| Deliverable | What It Contains | Source Questions |
|-------------|------------------|-----------------|
| **Positioning scorecard** | How each positioning component scored against target personas | All 7 questions |
| **Competitive alternative map** | What customers actually do today (not what you assume) | Q1, Q2 |
| **Value resonance ranking** | Which value propositions landed strongest | Q3, Q6 |
| **Market category feedback** | How customers naturally categorise you | Q4 |
| **Risk flags** | Where positioning assumptions don't match reality | Q5, Q7 |
| **Quotable insights** | Direct persona quotes for positioning docs | All responses |

### From a Messaging Testing Study

| Deliverable | What It Contains | Source Questions |
|-------------|------------------|-----------------|
| **Message performance ranking** | Win/loss rationale per variant | Q1, Q2, Q3 |
| **Clarity scorecard** | Which messages understood correctly vs misinterpreted | Q1, Q4 |
| **Language harvest** | Words and phrases that resonated (for copywriting) | Q6 |
| **Audience-message fit matrix** | Which message works for which persona type | Q1-Q3 cross-referenced with demographics |
| **Recommended primary message** | Data-backed winner with evidence | Q3 |
| **Messaging hierarchy** | Primary, secondary, tertiary messages | All responses |

### From a Competitive Intelligence Study

| Deliverable | What It Contains | Source Questions |
|-------------|------------------|-----------------|
| **Competitive perception matrix** | How targets view us vs each competitor | Q1, Q2 |
| **Landmine questions** | Questions that highlight competitor gaps (for sales) | Q3, Q4 |
| **Quick dismisses** | One-line rebuttals for competitor claims | Q4 |
| **Win themes** | Top 3-4 reasons customers choose us | Q2, Q5 |
| **Loss themes** | Top 3-4 reasons they'd choose competitor (with mitigation) | Q3, Q7 |
| **Switching trigger map** | Events that create competitive openings | Q6 |
| **Full battlecard** | One-page: Why We Win, Competitor Strengths, Landmines, Quick Dismisses | All responses |

### From a Pricing Study

| Deliverable | What It Contains | Source Questions |
|-------------|------------------|-----------------|
| **Price sensitivity band** | Acceptable range based on responses | Q1, Q2 |
| **Feature-tier recommendation** | What goes in Free vs Pro vs Enterprise | Q4, Q5 |
| **Packaging preference breakdown** | Freemium vs flat-rate vs usage-based | Q3 |
| **Price anchoring strategy** | How to frame pricing against alternatives | Q1, Q7 |
| **Willingness-to-pay by persona** | Segment-specific thresholds | Q2 cross-referenced with demographics |
| **Competitive price positioning** | Where you sit relative to alternatives | Q7 |

### From a GTM Validation Study

| Deliverable | What It Contains | Source Questions |

## Sales Enablement Factory

One study can feed an entire sales enablement kit:

| Sales Deliverable | Source Data | Format |
|-------------------|------------|--------|
| **Competitive battlecard** | Competitive study Q2-Q5 | One-page: Why We Win, Competitor Strengths, Landmines |
| **Objection handling guide** | Launch/positioning Q4, Q7 | Table: objection -> response with evidence |
| **Customer evidence quotes** | All responses contain quotable statements | Quote bank organised by theme |
| **One-pager** | Positioning Q4 + Launch Q5 | Customer-language product summary |
| **Pitch narrative** | Study insights on problems + value drivers | StoryBrand: customer problem -> guide -> plan -> success |
| **ROI framework** | Pricing Q1 + VoC Q3 | Quantified value framework |
| **Demo script** | Launch Q3 (first thing they'd try) | Feature-prioritised demo flow |

## Content Marketing Factory

One study can feed multiple content assets:

```
Ditto Study
    |-> Blog article (1,000-2,500 words, SEO/GEO optimised)
    |-> Infographic (key findings visualised)
    |-> Social thread (3-5 posts with quotable insights)
    |-> Email newsletter segment (study highlights)
    |-> Executive summary (1-page for C-suite)
    |-> Webinar/presentation content (slide-ready insights)
    |-> Press release (newsworthy findings)
```
